# utbrowser
A simple and naive web browser

**Requirements**

* [gumbo-parser](https://github.com/google/gumbo-parser) pure C99 HTML parser

* [SDL2](https://www.libsdl.org/)

* [cairo](https://www.cairographics.org/)

* [pango](https://www.pango.org/)

For MacOS
```
brew install sdl2
brew install cairo
brew install pango
```
